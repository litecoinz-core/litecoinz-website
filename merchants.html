Merchant Integration

LtzCore Merchant Services Toolkit

Merchants who wish to integrate with LitecoinZ can do so via the self-hosted HTTP API provided by the LtzCore toolkit. LtzCore is a fork of the industry standard BitCore toolkit for Bitcoin, and offers a streamlined method of confirming payments for web applications. LtzCore is built using the Node.js platform and integrates well with many modern web frameworks.
